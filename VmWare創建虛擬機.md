選擇了需要的磁碟檔案.iso

記憶體與CPU數量需視需求設定建議比本機小一半以下

等待開機開機須先進入Bios改成磁碟開機會觸發安裝流程

直到這些都完成後重開機即可使用環境

另外注意一個TCP概念在虛擬機中有一個
| 模式  | 英文名稱                          | 定義                             |
| --- | ----------------------------- | ------------------------------ |
| NAT | Network Address Translation   | 虛擬機透過主機共享網路對外上網，外部網路看不到虛擬機的 IP |
| 橋接  | Bridged Network (Bridge Mode) | 虛擬機直接掛在實體網路上，像一台獨立電腦，有自己的 IP   |
橋接是模擬一個實體網路只是跟你共用網線所以會廣播獲得一個屬於自己的網路IP與主機還有整個網域屬於同等級
而NAT則是在內部建立一個內網的概念，外部不會知道也無法Ping道內網但內網可以Ping出來
原因是內往封包出來實際上還是走本機的IP+連線時開設的POR口，比喻起來就是大樓內的房號所以回復的時候就是依照這些資料回傳回本機內往透過Port口收到
而橋接則是與外部實體網路相等所以互相通訊沒有問題
