OSI Model
│
├─ 第 7 層 Application Layer 應用層
│  ├─ HTTP
│  ├─ HTTPS
│  ├─ FTP
│  ├─ SMTP
│  ├─ DNS
│  ├─ DHCP
│  └─ Modbus TCP
│
├─ 第 6 層 Presentation Layer 表示層
│  ├─ 資料格式轉換
│  ├─ 編碼 Encoding
│  ├─ 加密 Encryption
│  ├─ 解密 Decryption
│  └─ 壓縮 Compression
│
├─ 第 5 層 Session Layer 會議層
│  ├─ 建立會話 Session Establishment
│  ├─ 維持會話 Session Management
│  └─ 結束會話 Session Termination
│
├─ 第 4 層 Transport Layer 傳輸層
│  ├─ TCP
│  │  ├─ Port Number
│  │  ├─ 連線建立 Three-way Handshake
│  │  ├─ 可靠傳輸
│  │  ├─ 重送機制
│  │  └─ 流量控制
│  │
│  └─ UDP
│     ├─ Port Number
│     ├─ 無連線
│     └─ 較低延遲
│
├─ 第 3 層 Network Layer 網路層
│  ├─ IP
│  │  ├─ IPv4
│  │  └─ IPv6
│  ├─ ICMP
│  ├─ Routing 路由
│  ├─ Router 路由器
│  ├─ Gateway 閘道
│  └─ IP Address
│
├─ 第 2 層 Data Link Layer 資料鏈結層
│  │
│  ├─ LLC 子層
│  │  └─ Logical Link Control 邏輯鏈結控制
│  │
│  └─ MAC 子層
│     ├─ Media Access Control 媒體存取控制
│     ├─ MAC Address
│     ├─ Ethernet Frame
│     ├─ ARP
│     ├─ VLAN
│     └─ Switch
│
└─ 第 1 層 Physical Layer 實體層
   ├─ 網路線
   │  ├─ Cat5e
   │  ├─ Cat6
   │  └─ Cat6A
   ├─ RJ45
   ├─ 光纖 Fiber
   ├─ 電訊號
   ├─ 光訊號
   ├─ 速度 Speed
   │  ├─ 100 Mbps
   │  ├─ 1 Gbps
   │  └─ 10 Gbps
   └─ 雙工模式 Duplex
      ├─ Half Duplex
      └─ Full Duplex
