| 狀態       | 意義                 |
| -------- | ------------------ |
| 電流流入元件正端 | P > 0，元件吸收功率：消耗或充電 |
| 電流流出元件正端 | P < 0，元件輸出功率：放電或發電 |
Wh:Wh = 功率 P × 時間 h ，能量：功率隨時間累積的結果通常都用正值累加，不顯示負號。
Import Energy = 輸入電能 / 流入累積電能 kWh
Export Energy = 輸出電能 / 流出累積電能 kWh

Import Power / Export Power  看瞬間功率方向
Import Energy / Export Energy 看累積電能
Net Energy = 淨電能 / 淨累積電量
公式=Net Energy = Import Energy - Export Energy
當數值為正表示充電大於放電
數值表示為負放電大於充電

自己的說法
簡單說我設備輸出電能對誰做功誰就是正功率而 如果是放電設備她不是在被做功而是反向向別人做功就是負功率

定義說法
功率正負號的正式定義：

對某一個被觀察設備而言，若電能流入該設備，則該設備的功率為正功率 +P。
對某一個被觀察設備而言，若電能流出該設備，則該設備的功率為負功率 -P。

也就是：

功率進入設備 = +P
功率離開設備 = -P

這個觀念稱為被動符號規則（Passive Sign Convention）。

在此定義下：

1. 負載消耗電能時：
   外部電源 → 負載
   功率進入負載，所以負載功率為 +P。

2. 電池充電時：
   PCS / 充電器 → 電池
   功率進入電池，所以電池功率為 +P。

3. 電池放電時：
   電池 → PCS / 負載
   功率離開電池，所以電池功率為 -P。

4. 發電機發電時：
   發電機 → 負載 / 系統
   功率離開發電機，所以以發電機本體來看，發電機功率為 -P。

5. 太陽能逆變器發電時：
   太陽能逆變器 → AC母線 / 負載 / 市電
   功率離開逆變器，所以以逆變器本體來看，逆變器功率為 -P。

重點：

+P 不是固定代表「供電」。
+P 是代表「對該設備而言正在吸收功率」。

-P 不是固定代表「異常」。
-P 是代表「對該設備而言正在輸出功率」。

所以必須先定義觀察對象，才能判斷 P 的正負號。

最簡單記法：

被供電、被充電、被消耗 = +P
供電、放電、發電輸出 = -P
